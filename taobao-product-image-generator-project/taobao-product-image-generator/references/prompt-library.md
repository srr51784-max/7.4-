# Prompt Library

Use these templates as building blocks. Preserve user-specified wording first.

## Blue Business Main Image

为淘宝生成一张1:1产品主图。产品居中放大展示，占画面65%-75%。背景使用蓝色简约商务风，白色与浅蓝渐变，少量科技线条。产品保持原始形状、角度、结构、材质和关键细节，不要改变产品本体。加入短卖点标签：品牌正版授权、高清实拍质感、精准适配车型。文字简短，不遮挡产品。

## Deep Blue Business Main Image

为淘宝生成一张1:1产品主图。画面采用深蓝色高级商务风，金属光影，专业汽配品牌视觉。产品主体清晰放大，边缘锐利，背景干净。只使用2-3个短标签，不要密集文字。

## Red Promotion Image

为电商活动生成一张1:1促销主图。红色喜庆促销风，适合周年庆、限时活动和咨询惊喜价。产品仍然是画面主体，活动文案要醒目但不盖住产品。避免廉价杂乱风格。

## Apple Minimal Style

生成苹果官网式极简产品图。大面积留白，干净白灰背景，高级光影，产品居中，精修质感。文字极少，只保留一句核心卖点或品牌标识。

## White Background Refinement

生成白底精修图，只展示产品本体。纯白背景，无文字、无图标、无包装盒、无场景元素、无额外配件。提升产品清晰度、光影和质感，但严格保持产品形状、角度、比例和结构。

## Detail Page 1440x1920

生成1440x1920px电商详情页竖图。顶部为短标题，中部大产品图，下部用2-3个重点说明。分区清楚，留白充足，一张图只表达一个卖点。产品要清晰放大，不要做成多图拼贴。

## Reference Image Editing

基于参考图进行改图。严格保持产品原始形状、角度、结构、材质和关键细节。只修改用户指定的背景、比例、包装、文字区域或颜色风格。不要生成新的产品形态，不要添加未提供的车标或品牌标志。
