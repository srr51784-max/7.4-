# Detail Image Workflow

Use this workflow when the user asks for 1440x1920px detail images or vertical product explanation images.

1. Default to `1440x1920`.
2. Build one image per selling point.
3. Use vertical hierarchy: header, product hero area, feature section, small footer.
4. Keep product clear and enlarged.
5. Do not create one collage containing all requested images.
6. Avoid repeated copy across a batch.
7. For automotive parts, prefer clean blue/white technical style unless the user requests another color.
8. For white-background detail shots, remove all layout elements and only show the product.
