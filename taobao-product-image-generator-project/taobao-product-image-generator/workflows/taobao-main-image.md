# Taobao Main Image Workflow

1. Confirm product category and platform from the user request.
2. Select output type `main` unless the user asks for detail page, white background, or editing.
3. Use `1:1` as default ratio.
4. Keep the product large and clear. Do not crop key parts.
5. Preserve product shape and reference perspective by default.
6. Select one style:
   - `blue-business` for clean automotive/product listings.
   - `deep-blue-business` for premium automotive listings.
   - `red-promo` for anniversary and promotional campaigns.
   - `apple-minimal` for high-end minimal commercial pages.
   - `white-clean` for product-only refinement.
7. Generate independent prompts for each requested image.
8. Ensure each image has different selling points.
9. Add store name only when requested.
10. Avoid warranty, official authorization, brand history, or original factory claims unless provided by the user.
